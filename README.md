## 快速部署数据库
>##### 使用命令行
>```shell script
>mysql> mysql -u root -p < deploy.sql
>```
>OR
>```shell script
>mysql> mysql -u root -p
>mysql> source deploy.sql
>```
>_(请将 `deploy.sql` 替换为正确的路径)_  
>_(必需使用 `root` 等超级权限用户执行 `deploy.sql` )_

>##### 使用相关 DBMA 执行SQL脚本（如 DataGrip 等）
